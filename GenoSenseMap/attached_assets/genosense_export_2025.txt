1. Deployment Frontend (Web App)
GitHub Pages
  1. Di repositori GitHub, buka Settings → Pages.  
  2. Pilih branch (misal: `main`) dan folder `/docs` atau `/dist` (hasil build).  
  3. Klik Save – aplikasi akan live di `https://<username>.github.io/GenoSense`.  
2. Deployment Backend (API)
 
  1. Buat file `Procfile` di root repositori:  
     ```txt
     web: python app.py   Jika menggunakan Flask
     ```  
  2. Install dependensi:  
     ```bash
     pip install gunicorn   Untuk Python
     ```  
  4. Login via CLI:  
     ```bash
     heroku login
     ```  
  5. Deploy dari GitHub:  
     ```bash
     heroku create genosense-api
     git push heroku main
     ```  
  6. Akses API di `https://genosense-api.herokuapp.com`.  


 3. Database
